# 함수
> 함수(function)란 하나의 로직을 재실행 할 수 있도록 하는 것으로 코드의 재사용성을 높여줍니다.
- 문법
```javascript
function 함수명( [인자...[,인자]] ){
코드
return 반환값
}
```
## 함수호출
- 예제
```javascript
function numbering(){
document.write(1);
}
numbering();
```
